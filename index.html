mocli

Alcor command line utility

pip install mocli

Dev/Personal Install

Install Lmod

# Install Lmod to the given location
alcor init /opt/alcor

# Modify bash to automatically source alcor bash profiles
alcor activate --auto

# Reload environment
exec bash

# Show a list of available modules
module spider

Install New Modules

alcor install lz4

module load lz4

# all the paths are set we can compile programs with lz4
tee lz4_main.c << END
#include "lz4.h"

int main(int argc, const char* argv[]){
    return 0;
}
END

gcc lz4_main.c -o lz4_main -llz4

Install Apt packages

Warning

This is not guaranteed to work. Package might hardcode library path or expect a specific system setup.

alcor aptinstall libeigen3-dev

module load libeigen3-dev

tee eigen_main.c << END
#include "eigen3/Eigen/Core.h"

int main(int argc, const char* argv[]){
    return 0;
}
END

gcc eigen_main.c -o eigen_main

Cluster Installation

  • The main installation is readonly

  • Update /etc/profile so every user connecting to the cluster gets modules

  • user can install their own package locally in their home
